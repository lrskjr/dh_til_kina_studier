# Digital Humaniora, tekstanalyse og Kinastudier

Fire notebooks med eksempler på, hvordan Folkets Dagblad kan analyseres vha. af Python. Indholdet er udviklet over tid i samarbejde med Bo Ærenlund Sørensen, Institut for Tværkulturelle og Regionale Studier, Københavns Universitet.

- `1 term frrequency.ipynb` finder de mest hyppige ord i et tekstudvalg.
- `trends.ipynb` viser udviklingen i et søgeords forekomst over tid.
- `pos tagging and context reading.ipynb` undersøger både sproglige mønstre og konkrete tekstkontekster.
- `spacy_chines_setup.ipynb` viser, hvordan kinesisk NLP kan sættes op og bruges i praksis.
